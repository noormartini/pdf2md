## Mathematical Formulas

This page demonstrates formula handling.

### Inline Formulas

The relationship between precision (P) and recall (R) can be expressed as the F1 score:

F1 = 2 × (P × R) / (P + R)

### Display Formulas

The accuracy metric is calculated as:

    accuracy = (TP + TN) / (TP + TN + FP + FN)

Where:
- TP = True Positives
- TN = True Negatives
- FP = False Positives
- FN = False Negatives

### Complex Expression

The cross-entropy loss function:

    L = -Σ(y_i × log(ŷ_i))
