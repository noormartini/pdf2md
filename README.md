# Multimodal conversion of PDF documents to Markdown for fast and accurate format conversion

## Thesis Info

- **Nächster Termin:** Mittwoch, 25.04 um 10:00 Uhr, Raum A102
- **Requirements-Datei** mitbringen

---

## Ziel

- Bachelorarbeit / Paper PDFs in Markdown umwandeln
- Effizientes, kompaktes Python-Skript

---

## Features

### Must-Have
- Inhaltsverzeichnis korrekt übernehmen
- Quelltext / Code-Blöcke richtig erkennen
- Spracherkennung (DE / EN)
- Grafiken ausschneiden und in Markdown einbinden

### Nice-to-Have
- **Schnelle Variante:** Nur Text (keine Bilder/Formeln)
- **Erweiterte Variante:** Mit Bildern und Formeln
- Formeln in LaTeX umwandeln

## Tools & Technologien

- **LM Studio** — LLM lokal laufen lassen (HS-Rechner prüfen, ob es direkt darauf geht)
